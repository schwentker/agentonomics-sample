# Example Goal: Create a Simple Python Project

Create a simple Python calculator project with the following requirements:

1. Create a `calculator.py` file with basic arithmetic functions:
   - add(a, b)
   - subtract(a, b)
   - multiply(a, b)
   - divide(a, b)

2. Create a `test_calculator.py` file with unit tests for each function

3. Create a `README.md` file documenting the calculator usage

4. Ensure all functions have proper docstrings and type hints
